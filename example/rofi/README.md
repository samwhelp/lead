
# lead usage example - rofi

## Develop Environment

* Ubuntu 18.04

## Help

``` sh
$ make
```

or

``` sh
$ make help
```


## Install package (rofi)

``` sh
$ make prepare
```

* [rofi](https://packages.ubuntu.com/bionic/rofi)


## Install config file (lead and rofi)

``` sh
$ make install
```


## Start lead

``` sh
$ make start
```


## Test lead

move mouse to bottom or bottomLeft or bottomRight.


## Stop lead

``` sh
$ make stop
```


## Remove config file (lead and rofi)

``` sh
$ make remove
```
