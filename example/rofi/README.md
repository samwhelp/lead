
# lead usage example - rofi


## Help

``` sh
$ make
```

or

``` sh
$ make help
```


## Install package (rofi)

``` sh
$ make prepare
```


## Install config file (lead and rofi)

``` sh
$ make install
```


## Start lead

``` sh
$ make start
```


## Test lead

move mouse to bottom or bottomLeft or bottomRight.


## Stop lead

``` sh
$ make stop
```


## Remove config file (lead and rofi)

``` sh
$ make remove
```
